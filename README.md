# treestats

R package that combines functions to calculate summary statistics on phylogenies.
Includes fast C++ code to calculate some of these statistics, including:
  - Aldous Beta
  - Sackin index
  - Colless index
  - nLTT
  - Gamma
  - Phylogenetic Diversity
  - RPANDA::spectR
  - ape::branching.times
  - DDD::phylo2L
  - DDD::L2phylo
